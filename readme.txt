== Description ==
MK Post and Page Excerpts Widgets :  Creates widgets that display excerpts from posts or pages in the sidebar. You may use 'more' links, show featured image, set excerpt length of the post or page.

Created By: Manoj Rana
Email : manoj.rana91986@gmail.com

== Installation ==

1. Upload the plugin to your /wp-content/plugins/ directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Go to your widgets area and you can see the widgets appearing there.

4. Drag drop the widgets to your sidebar and you can see the result.


== Frequently Asked Questions ==
= Can I show posts/page featured image = 
Yes, there is option to show/hide the post/page featured image.
